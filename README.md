# Machine-learning-project
## Overview

This repository contains multiple Machine Learning projects implemented using Python and Jupyter Notebook. The projects cover various supervised learning algorithms and demonstrate data preprocessing, model building, training, evaluation, and prediction techniques.
Projects Included
Decision Tree
Random Forest
K-Nearest Neighbors (KNN)
Logistic Regression
Multiple Linear Regression
Gradient Boosting
Naive Bayes
Support Vector Machine (SVM)
## Technologies Used
Python
Jupyter Notebook
NumPy
Pandas
Matplotlib
Seaborn
Scikit-Learn
